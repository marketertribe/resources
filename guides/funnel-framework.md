# The Funnel Building Framework

A step-by-step approach to building marketing funnels that convert.

## The Core Funnel Types

### 1. Lead Magnet Funnel (Free → Email List)
```
Traffic → Landing Page → Thank You Page → Email Sequence
```
**Best for**: Building an email list, validating an offer
**Key metric**: Opt-in rate (target: 25-40%)

### 2. Tripwire Funnel (Free → Low-Ticket → Upsell)
```
Traffic → Landing Page → $7-27 Offer → Order Bump → Upsell → Thank You
```
**Best for**: Acquiring customers at breakeven, building a buyer list
**Key metric**: Cost per acquisition vs. average order value

### 3. Webinar Funnel (Education → High-Ticket)
```
Traffic → Registration → Webinar → Application/Sales Page → Follow-Up
```
**Best for**: Selling $200+ products or services
**Key metric**: Registration rate, attendance rate, conversion rate

### 4. Application Funnel (Qualify → High-Ticket Service)
```
Traffic → Landing Page → Application Form → Calendar Booking → Sales Call
```
**Best for**: Services, coaching, consulting ($1,000+)
**Key metric**: Application quality, show rate, close rate

## The 3-Step Funnel Audit

### Step 1: Traffic Quality
- Where are visitors coming from?
- Is the traffic cold, warm, or hot?
- Does the traffic source match the offer?

### Step 2: Conversion Points
- Landing page opt-in rate (benchmark: 25-40%)
- Sales page conversion (benchmark: 1-5%)
- Cart completion rate (benchmark: 60-80%)

### Step 3: Follow-Up Sequences
- Are non-converters getting follow-up emails?
- Is there a clear ascension path?
- Are you segmenting based on behavior?

## Common Funnel Mistakes

1. **Too many steps**: Every click loses 20-30% of visitors
2. **Weak headline**: You have 3 seconds to hook attention
3. **No urgency**: Without a reason to act now, they won't
4. **Missing social proof**: Testimonials aren't optional
5. **Ignoring mobile**: 60%+ of traffic is mobile
