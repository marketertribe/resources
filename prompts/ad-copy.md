# AI Ad Copy Prompts

Prompts for generating high-converting ad copy across platforms.

## Facebook/Instagram Ads

### Problem-Agitate-Solve
```
Write Facebook ad copy using the PAS framework:
Product: [PRODUCT]
Target audience: [AUDIENCE]
Main pain point: [PAIN]
Format: Primary text (max 125 words), Headline, Description, CTA button text.
Provide 3 variations.
```

### Testimonial-Based Ad
```
Convert this customer testimonial into Facebook ad copy:
Testimonial: "[PASTE TESTIMONIAL]"
Product: [PRODUCT] at [PRICE]
Format: Lead with the most compelling quote, add context, include CTA.
Provide primary text, headline, and description.
```

## Google Ads

### Search Ad Copy
```
Write Google Search ad copy for [PRODUCT/SERVICE].
Target keyword: [KEYWORD]
Headline 1 (30 chars): Include keyword
Headline 2 (30 chars): Benefit-focused
Headline 3 (30 chars): CTA-focused
Description 1 (90 chars): Value proposition
Description 2 (90 chars): Social proof or urgency
Provide 5 variations.
```

### Performance Max Headlines
```
Generate 15 headlines for a Google Performance Max campaign:
Business: [BUSINESS]
Product: [PRODUCT]
USP: [UNIQUE SELLING PROPOSITION]
Mix: 5 benefit-focused, 5 feature-focused, 5 CTA/urgency-focused.
Each under 30 characters.
```

## YouTube Ads

### Pre-Roll Script (15 seconds)
```
Write a 15-second YouTube pre-roll ad script for [PRODUCT].
First 5 seconds: Hook that prevents skipping.
Next 7 seconds: Core value proposition.
Last 3 seconds: Clear CTA.
Target audience: [AUDIENCE]
```
