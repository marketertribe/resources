# AI Funnel Copy Prompts

Prompts for every stage of your marketing funnel.

## Landing Page Copy

### Opt-In Page
```
Write landing page copy for a free [LEAD MAGNET TYPE] about [TOPIC].
Target audience: [AUDIENCE]
Headline: Benefit-driven, under 10 words
Subheadline: Expand on the headline, address the pain point
Bullet points: 3-5 specific things they'll learn/get
CTA button text: Action-oriented (not just "Submit")
Tone: Direct, valuable, no hype
```

### Sales Page
```
Write a long-form sales page for [PRODUCT] at [PRICE].
Target audience: [AUDIENCE]
Sections needed:
1. Headline + subheadline (curiosity + benefit)
2. Problem section (3 pain points)
3. Solution introduction
4. What's included (feature → benefit format)
5. Social proof section
6. FAQ (5 common objections)
7. Final CTA with urgency
Tone: Conversational, confident, zero hype. Write like a smart friend recommending something.
```

### Thank You / Confirmation Page
```
Write thank you page copy for someone who just [PURCHASED/SIGNED UP] for [PRODUCT].
Include: Confirmation of what they'll receive, next steps, one upsell mention (soft), community invitation.
Tone: Excited but professional.
```

## Webinar Funnel

### Webinar Registration Page
```
Write registration page copy for a free webinar: "[WEBINAR TITLE]"
Host: [NAME], [CREDENTIALS]
Date/time: [DATE]
Include: What they'll learn (3 bullet points), who it's for, urgency element.
```

### Webinar Follow-Up Email
```
Write a follow-up email for webinar attendees who didn't purchase.
Product pitched: [PRODUCT] at [PRICE]
Include: Recap key points, address top objection, limited-time offer.
Tone: Helpful, not desperate.
```
