# AI Social Media Prompts

Prompts for generating engaging social media content across platforms.

## Twitter/X Thread Prompts

### Educational Thread
```
Write a 7-tweet thread about [TOPIC] for [TARGET AUDIENCE].
Format: Hook tweet (curiosity gap) → 5 value tweets (one tip each) → CTA tweet.
Each tweet: Under 280 characters. Use line breaks for readability.
Tone: Authoritative but approachable. No emojis in hook tweet.
```

### Contrarian Take
```
Write a Twitter thread challenging the common belief that [COMMON BELIEF] in [INDUSTRY].
Format: Bold contrarian hook → 3-4 supporting arguments with evidence → Alternative approach → CTA.
Tone: Confident, backed by reasoning, not just provocative for clicks.
```

## LinkedIn Post Prompts

### Story-Driven Post
```
Write a LinkedIn post about [LESSON LEARNED] using this framework:
- Hook: One surprising line (under 15 words)
- Story: Brief personal anecdote (3-4 sentences)
- Lesson: The takeaway (2-3 bullet points)
- CTA: Engage the audience with a question
Length: 150-200 words. Use short paragraphs (1-2 sentences each).
```

### Value Post
```
Write a LinkedIn post sharing [NUMBER] tips about [TOPIC] for [AUDIENCE].
Format: Hook line → Numbered tips (one line each) → Brief closing thought → Question to engage.
Tone: Professional but human. No corporate jargon.
```

## Instagram Caption Prompts

### Carousel Post Outline
```
Create an Instagram carousel outline about [TOPIC].
Slide 1: Attention-grabbing title (5-7 words max)
Slides 2-7: One key point per slide with brief explanation
Slide 8: CTA slide
Caption: Write a 100-word caption that expands on the carousel topic.
Include: 5 relevant hashtags.
```

### Reel Script
```
Write a 30-second Instagram Reel script about [TOPIC].
Hook (first 3 seconds): Pattern interrupt or bold statement.
Body: 3 quick tips or steps.
CTA: Tell them what to do next.
Format: Include visual direction notes in [brackets].
```

## Facebook Ad Copy

### Lead Generation Ad
```
Write Facebook ad copy for [PRODUCT/SERVICE] targeting [AUDIENCE].
Format: Primary text (125 words max), Headline (5-7 words), Description (one line).
Angle: [PAIN POINT] → [SOLUTION] → [CTA]
Include: 3 variations with different hooks.
```
